# ConfusedPreset
This preset is intended for phycological thillers, horror, and slice-of-life role play for GLM thinking.
The preset includes toggles to tailor as needed outside of fantasy & sci-fi Role Play.
This is token heavy (1,350) but it does follow the rules very well.
Testing on GLM 5 Thinking via NanGPT PAYG.  GLM Direct has not been tested yet.
